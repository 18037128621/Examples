90后网民的用户形态与上网习惯调查(一）

这是我最近颇感兴趣的一个话题，90后网络用户作为社会的一个焦点，在网上非常的活跃。他们的网站访问行为较总体网络用户的总体存在一定的差异。你可以理解为“代沟”！70后淡出互联网、80后是互联网用户的最大人群，但是随着年龄的增长，“老婆、孩子、车子、房子、贷款”让80后很忙碌。90后恰逢风华正茂，“吃喝玩乐”之正当时。90后将取代80后的网友峰值地位，渐渐成为网络主体。

这就是研究90后形态的核心所在，我们是做网络的，我们必需及时了解用户动向和发展趋势，猜透他们的喜好，才能更好的服务用户。

90后网民与网民总体的差异性：

 1、90后网民对购物、娱乐及视频等类型网站的访问偏好较强

 可见这类人群还是偏娱乐化，喜欢看视频，看娱乐八卦。 而作为未来的消费主力军，他们的网上购物习惯也正在建立起来。但是在购物网站中，反而是中小型的垂直购物网站比较成功的吸引了90后，比如千寻服饰，一家网，唯品会等等。他们共同的特点就是流行服饰，当然也是女性用户较多。

 2、垂直型的中小购物网站，成功的吸引了90后

 其中前3类都是90后很集中的，但是浏览深度有所差异，在营销选择的时候需要有所侧重。后3类是90后活动并不集中的媒体，在常规的大网站90后浏览时间都偏短，可能90后对成人社会的很多问题并不感兴趣，但是又有好奇，最后浅尝则止。
