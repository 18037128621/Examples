孕妇使用电脑有哪些注意事项

电脑病，每一个工作女性日常都该防范，更何况到了孕期？孕妇使用电脑需要注意什么？您不妨在准备长时间工作在电脑前时，选好设备，坐好姿势，为你的身体带来更多舒适，减轻各种隐疾的发生。

1、孕妇每周接触电脑时间不超过20小时
要防止长时间坐位引起盆腔血液滞留不畅，做到张弛有度；要注意电脑与座椅坐姿的高低配合，让胎儿健康发育。
2、注意劳逸结合，防止肌腱劳损
长时间操作电脑会导致手指关节、手腕、手臂肌肉、双肩、颈部、背部等部位出现酸胀疼痛，因此，孕妈妈们在工作一小时后应休息10分钟，或者做做工间操。
3、要注意用眼卫生
眼睛与文稿、眼睛与屏幕的距离应保持在50厘米以上，最好采用光下视20度的视角。工作时，应在面部及双手涂抹防辐射的护肤油。

4、长期从事电脑操作的孕妇，应多吃一些新鲜的蔬菜和水果。增加维生素A、B1、C、E的摄入。
为预防角膜干燥、眼干涩、视力下降、甚至出现夜盲症等，用电脑孕妇应多吃些富含维生素A的食物，如豆制品、鱼、牛奶、核桃、青菜、大白菜、西红柿、空心菜及新鲜水果等。维生素C可以有效地抑制细胞氧化。维生素E主要作用是：降低胆固醇，清除身体内垃圾，预防白内障。核桃和花生中含有丰富的维生素E。维生素B1可以加强神经细胞的营养，缓解神经的紧张状态。 5、每天泡点绿茶
　　茶叶中的脂多糖，可改善机体造血工能。人体注入脂多糖后，在短时间内即可增强机体非特异性免疫力。茶叶还能防辐射损害。
